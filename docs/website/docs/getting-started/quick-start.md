# Quick Start

Get the ScoreBook app running in 5 minutes.

## Start Backend

```bash
cd backend
npm run dev
```

Backend runs at: `http://localhost:3000`

## Start Mobile App

### Web

```bash
cd mobile
npx expo start --web
```

Opens at: `http://localhost:8081`

### iOS

```bash
cd mobile
npx expo run:ios
```

### Android

```bash
cd mobile
npx expo run:android
```

## Verify It Works

1. Open the app (web/iOS/Android)
2. You should see the Cricket tab selected
3. Navigate through different sports
4. Check live, upcoming, and completed matches

That's it! You're ready to start developing.
